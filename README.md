react-issue-ganttchart
===================

This is a react gantt-issue-ganttchart.

### How to start

 - clone the repository or download files
 - install dependencies
~~~
yarn install
~~~

 - run server
~~~
yarn start
~~~

- build and deploy for Pages
~~~
yarn deploy
~~~
